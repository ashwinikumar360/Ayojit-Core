# Privacy Policy

**Ayojit Intelligence**
Last Updated: June 16, 2026

## 1. Data We Collect

| Data Type | Purpose | Retention |
|-----------|---------|-----------|
| Email address | Authentication | Account lifetime |
| Full name | Display in dashboard | Account lifetime |
| Phone number | Optional profile field | Account lifetime |
| Usage logs | Quota enforcement, analytics | 90 days |
| Generated content URLs | History display | Account lifetime |
| API key hashes | Authentication | Until revoked |

## 2. How We Use Your Data

- Authenticate your identity via Supabase Auth
- Enforce daily quota limits per app
- Display your usage history in the dashboard
- Send transactional emails (welcome, quota alerts, receipts)

## 3. Data We Do Not Collect

- Voice recordings (Kisan Voice AI processes audio in-memory only)
- Document content (DocPatram generates documents server-side, does not store input text)
- Payment card details (handled entirely by Dodo Payments)
- Browsing history or third-party tracking cookies

## 5. DPDP Act Compliance

Ayojit Intelligence complies with the Digital Personal Data Protection Act, 2023 (India):

- We process personal data only with consent (account creation)
- Data is used only for stated purposes
- Users may request deletion of their data
- We implement reasonable security safeguards
